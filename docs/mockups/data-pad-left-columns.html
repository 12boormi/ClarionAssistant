In the proposed layout the type column starts just past the longest name and the picture column starts just past the longest type — both left-aligned, so each column lines up vertically under its longest entry. Column widths auto-size to content (CSS grid max-content), and nesting indent is absorbed by the name column without shifting the type/picture columns. Gap between columns is tunable (currently 22px).
